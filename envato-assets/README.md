# Envato Asset Intake

This folder is the local intake area for paid Envato downloads used by the QCSS network and security website tool.

Do not commit downloaded ZIPs, raw design files, stock media, source videos, font files, or license PDFs unless we intentionally process and move a specific asset into the application. The repo tracks only this README, the checklist, and the asset register.

## Folder Map

- `01-core-nextjs-theme`: primary ThemeForest/Elements website theme reference.
- `02-admin-dashboard`: dashboard/admin template for the private command center.
- `03-ui-kit-figma`: UI kit, Figma files, design systems, and component references.
- `04-3d-cyber-network-graphics`: 3D renders, Spline/Blender models, tech visuals, cloud/security graphics.
- `05-icons-infographics`: icons, infographic systems, diagrams, and service illustration packs.
- `06-presentations-sales-decks`: proposal decks, course decks, audit decks, and webinar slide templates.
- `07-lead-magnet-pdfs`: brochure, checklist, whitepaper, report, and guide templates.
- `08-social-video-campaigns`: reels, video intros, LinkedIn carousel templates, and campaign creatives.
- `09-email-templates`: responsive email and newsletter templates.
- `10-codecanyon-review`: scripts/plugins to inspect only; do not integrate without security and license review.
- `licenses`: local license certificates, invoices, and Envato registration notes.
