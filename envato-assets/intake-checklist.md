## License Hygiene

For Envato Elements, create/register each item for the same QCSS project:

`QuantumCrafters Studio Network & Security Growth Website`

For ThemeForest and CodeCanyon, save the separate purchase/license files in `licenses`.

Before implementation, each downloaded package must pass:

- license file exists or registration is recorded;
- source URL is added to `asset-register.csv`;
- framework/dependency list is inspected;
- demo images/fonts/third-party assets are checked;
- no credentials, tracking IDs, or author demo endpoints are copied into production;
- source package is kept in `envato-assets`, while only selected processed assets enter `src` or `public`.
