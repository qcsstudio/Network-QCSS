The home-page Envato band now uses the new data-center/network command illustration as a client-facing topology visual and the 3D security objects as four practical service cards:

- Infrastructure Operations
- Security Controls

The section copy was rewritten to avoid exposing internal design or marketing strategy. It now speaks to the buyer's problem: scattered symptoms, unclear exposure, missing evidence, and the need for a practical next action.

## Deferred Assets

Deferred for performance or phase fit:

- Large `.mov` cyber animation files: visually strong, but most are 70 MB to 160 MB each and should be transcoded/compressed before web use.
- Raw `.aep`, `.blend`, `.fbx`, `.obj`, `.eps`, `.psd`, and presentation sources: useful as design source material, but not appropriate to publish directly.
- Full Envato theme source trees: use for layout and interaction reference only. The production site should remain a custom QCS Next.js product.
- Admin dashboards: keep for the next private command-center phase covering leads, funnel metrics, tracking, campaign performance, and sales views.
